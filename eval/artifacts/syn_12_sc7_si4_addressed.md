# SSP Excerpt — SC-7 / SI-4 only

## SC-7 Boundary Protection

External system boundaries are monitored and controlled by perimeter firewalls,
DMZ proxies, and an IDS sensor. The boundary components are: edge firewall,
DMZ proxy. The monitoring mechanism is the SIEM.

## SI-4 System Monitoring

The system is monitored to detect attacks. Monitoring objectives include
unauthorized access and lateral movement. The alerting mechanism is SIEM-driven
notifications to the SOC.
